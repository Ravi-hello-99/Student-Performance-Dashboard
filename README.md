# Student Performance Analytics Dashboard

## Project Overview
An end-to-end data analytics project built using Microsoft SQL Server and Power BI Desktop.

## Tools Used
- Microsoft SQL Server (MSSQL)
- Power BI Desktop
- SQL (DDL, DML, Views, Joins)

## Database Design
5 tables with proper relationships:
- Students
- Subjects
- Exams
- Marks
- Attendance

## Dashboard Features
- Grade distribution (Donut Chart)
- Average marks by Department (Bar Chart)
- Midterm vs Final comparison (Column Chart)
- Attendance % by Student (Bar Chart)
- Dynamic Department Slicer filter

## Files
| File | Description |
|---|---|
| Dashboard.pbix | Power BI Dashboard file |
